\section*{Dev Example Resume}

This is the bundled development resume template used when resumeclaw starts
without a configured provider or template directory.

\begin{itemize}
  \item Replace this content with your own resume details.
  \item Set \texttt{RESUME\_TEMPLATE\_DIR} to use a real LaTeX template.
  \item Configure \texttt{LLM\_PROVIDER} and the matching API key to use a real model.
\end{itemize}
